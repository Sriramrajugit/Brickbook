### Troubleshooting

**Certificate Error Persists?**
- Clear browser cache and cookies
- Hard refresh (Ctrl+Shift+R)

**Port Already in Use?**
- Kill process: `taskkill /F /IM node.exe`
- Restart: `npm run dev:https`

**Need HTTP for Testing?**
- Run `npm run dev` instead (no HTTPS)
